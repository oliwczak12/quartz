## Klasa Math

Importowanie klasy Math
```java
import static java.lang.Math.*;
```

### Przydatne stałe i metody 

Metody:

```Java
Math.sqrt(wartośc) - pierwiastek kwadratowy
Math.pow(wartość, potęga) - potęgowanie 
```

Stałe:

```java
Math.PI - wartość liczby pi
```
